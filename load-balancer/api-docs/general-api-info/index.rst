=======================
General API Information
=======================

The |product name| API is implemented using a RESTful web service interface.
|product name| shares a common token-based authentication system with other
products in  the Rackspace Cloud suite. This system enables seamless access
between Rackspace products and services.

.. note::
    All requests to authenticate against and operate the service are
    performed using SSL over HTTP (HTTPS) on TCP port 443.
